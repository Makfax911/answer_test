Здравствуйте, коллеги! Буду рад представить курс Java Уровень 1. Нам потребуется Java SE Development Kit (JDK - можно загрузить по адресу http://www.oracle.com/technetwork/java/javase/downloads/2133151) и ваш любимый текстовый редактор (рекомендую Notepad++ https://notepad-plus-plus.org/). Скачиваете и устанавливаете JDK и проверяете его работоспособность двумя командами (в командной строке, в Windows - через cmd):

C:\>java -version
C:\>javac -version

После установки JDK надо настроить "переменные среды", хорошее объяснения есть по ссылке http://study-java.ru/uroki-java/urok-1-ustanovka-java/comment-page-3/ Если у вас 64 битная версия Windows, то стоит ставить 64 версию JDK. Будут неясности, проблемы при установке - пишите, спрашивайте.
