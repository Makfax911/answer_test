IntelliJ IDEA для новичков: советы и секреты

Введение

IntelliJ IDEA — интегрированная среда разработки программного обеспечения для многих языков программирования, в частности Java, JavaScript, Python, разработанная компанией JetBrains. Появилась в январе 2001 года и быстро приобрела популярность как первая среда для Java с широким набором интегрированных инструментов для рефакторинга.

Начиная с версии 9.0, существуют две Edition: Community и Ultimate. Community Edition является полностью свободной версией, в ней реализована полная поддержка Java SE. Ultimate Edition, доступная под коммерческой лицензией, имеет поддержку Java EE и Spring.

На вебинаре мы рассмотрим основные приёмы работы с IDEA плюс ряд полезных свойств и трюков для ускорения работы. Необходимо иметь установленный JDK 8, желательно также скачать и установить саму IDEA Community Edition (https://www.jetbrains.com/idea/download/).

1. Запуск и создание первого проекта
    - обратить внимание на выбор JDK, пропустить template
    - поговорить о пакетах, для начинающих они не нужны
    - выбор имени и местоположения проекта
    - содержание папки проекта, папка src
    - настройка шрифтов - File | Settings | Editor | Font

2. Основы работы с кодом
    - создать класс - RightClick | New | Java Class | HelloWorld
    - обратить внимание на общие правила именования классов
    - поговорить об именовании классов для домашних заданий
    - psvm, sout - сокращения, но важно тренировать пальцы
    - комментарии в Java, три вида (//, /* */, /** */)
    - javadoc комментарий, аннотации @author и @version
    - запуск кода - RightClick | Run, консоль с результатами, перезапуск
    - внешний вид кода, переформатирование - Code | Reformat Code

3. Папка проекта - более детальный взгляд
    - где находится исходный код /src
    - где находятся скомпилированные классы /out/Production/<ProjectName>
    - работа в Terminal - View | Tool Windows
    - компиляция и запуск в терминале

4. Продвинутая работа с кодом
    - изменение имени класса - RightClick | Refactor | Rename...
    - работа в режиме отладки, точки останова (цикл)
    - импорт библиотек (класс Random, java.util)
    - генерация конструктора (на примере класса Cat)
    - генерация геттеров и сеттеров
    - генерация метода toString
