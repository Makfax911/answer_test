Тестовое задание

Требуется разработать приложение на связке технологий java + spring + jsf. Приложение должно быть разработано на основе данного spring boot приложения: https://bitbucket.org/kirill-salyamov/spring-boot-jsf/overview.

Необходимо реализовать сервис, который будет выводить товары из базы данных:
1. Вывод списка товаров на страницу
2. Сортировка товаров 
3. Фильтры товаров:
3.1. Задаваемый диапазоном фильтр по цене товара
3.2. Фильтр по местонахождению товара, задаваемый списком с checkbox-ами
3.3. Автоматическое обновление содержимого таблицы при изменении фильтров
4. Параметры сортировки списка должны сохраняться (например, в cookies)

Товар состоит из:
* названия
* цена
* местонахождение товара

В качестве БД должна использоваться HSQLDB.
