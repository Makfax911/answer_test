Тестовое задание Java:

public interface IProduct {
    int getPrice();
    void sell();
}

У вас есть интерфейс продукта IProduct, вам необходимо реализовать его для двух продуктов: яблоки и вино.

Продукт может дешеветь с каждым определенным проданным экземпляром, либо, наоборот, дорожать.
Информация об этом хранится в справочной таблице базы данных SQL.

Например, яблоки с каждым сотым проданным экземпляром дешевеют, а вино наоборот дорожает после каждого двадцатого проданного экземпляра.

Напишите программу, которая будет принимать на вход количество яблок и вина которое нужно продать и считает их сумму стоимости.

История запросов сохраняется в журнале, который так же располагается в таблице базы данных SQL.

В программе имеется возможность редактирования справочника продуктов.

Необходимую структуру базы данных необходимо предложить самостоятельно. Детализируйте реализацию только ключевых методов.

Реализация пользовательского интерфейса не обязательна (в этом случае вместо реализации методов взаимодействия с UI укажите в них соответствующие комментарии)

По выполнению обязательно укажите количество потраченного времени.
