Учебный план для персонального обучения

Цель: обучение основам программирования на Java
Книга: Java программирование для начинающих, Майк МакГрат

1. JDK 8, редактор кода, HelloWorld, компиляция и запуск. Класс и методы, вывод в консоль. Комментарии. Чтение ошибок в консоли. Переменные, типы переменных (int, float, String), присвоение значений, арифметические операции, конкатенация строк.

    Чтение книги: стр.12-17 (JDK+HelloWorld), стр.18-22 (переменные), стр.28-31 (операции с переменными)
    Практика:
    1. Набрать класс HelloWorld, скомпилировать, запустить (класс: HelloWorld)
    2. Набрать код из глав про переменные и арифметические операции (классы: FirstVariable, DataTypes, Constants)
    Мини-статьи:
    1. Первый шаг: JDK
    2. Второй шаг: редактор + HelloWorld
    3. Третий шаг: Игры с переменными
