Введение в Java

Пройдя этот 6-ти недельный вводный курс по Java, вы:
- Освоите базовые конструкции языка, основы ООП и основы Swing
- Напишете несколько простых игр, как консольных, так и графических
- Получите базовую подготовку для продолжения изучения языка

1. Знакомимся с Java
Вебинар 1. Базовые конструкции, переменные, условия, циклы
• установка JDK, HelloWorld, компиляция и запуск
• переменные, типы, строгая типизация
• условия if-else, тернарный оператор, выбор swith
• цикл со счётчиком for, по условию while, do-while
• импорт классов из пакетов, классы Random и Scanner
• консольная игра «Угадай число»

2. Продолжаем знакомство с Java
Вебинар 2. Массивы и коллекции, методы класса
• массивы и коллекции
• методы, параметры, возвращаемые значения, перегрузка
• статические и нестатические методы, основы ООП
• консольная игра «Крестики-нолики»

3. Введение в ООП
Вебинар 3. Классы и объекты, инкапсуляция, наследование, Swing
• класс, поля и методы, конструкторы, создание объектов
• инкапсуляция, модификаторы, геттеры и сеттеры
• наследование
• полиморфизм, интерфейсы

4. Введение в Swing
Вебинар 4. Основы работы с графической библиотекой Swing
• простое оконное приложение на базе JFrame
• размещение компонентов, менеджеры компоновки
• работа с кнопками, обработка нажатий
• обработка нажатий на клавиши, работа с мышью
• рисование

5. Закрепляем знания ООП и Swing на практике
Вебинар 5. Практическое применение ООП и Swing
• пишем игру «Лопни шарик», используя ООП и Swing

6. Финальный проект курса
Вебинар 6. Пишем игру «Змейка»
