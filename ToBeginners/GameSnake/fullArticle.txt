Пишем игру «Змейка» на Java

Введение

Java – объектно-ориентированный язык программирования высокого уровня, разработанный компанией Sun Microsystems. Впервые был официально представлен в 1995 году. В настоящий момент — один из самых популярных языков программирования. Позволяет вести разработку в четырех направления: настольные системы (desktop), интернет (web), мобильные системы (Android), встроенные системы (embedded systems).

Исходный код на Java компилируется в байт-код, выполняемый виртуальной машиной Java (JVM), которая существует для всех платформ: Windows, Linux, Mac.

Инструменты

Первый шаг подготовки к работе: скачать и установить JDK (Java Development Kit) – бесплатно распространяемый комплект разработчика Java. Он содержит компилятор (javac), стандартные библиотеки классов, примеры, документацию, различные утилиты и исполнительную систему Java (Java Runtime Enviroment, JRE), её также называют Java Virtual Machine (JVM).

При этом нет необходимости запоминать или записывать какой-то интернет адрес, достаточно уметь пользоваться поисковой системой Google. В строку поиска google.com вводим запрос: “jdk 8 download”, первая же строка поисковой выдачи даёт ссылку на страницу загрузки от oracle.com. Переходим, находим радио-кнопку “Accept License Agreement”, щелкаем по ней, соглашаясь с лицензией. Выбираем в колонке “Download” пакет согласно платформе (Window, Linux, Mac), cкачиваем и устанавливаем.
