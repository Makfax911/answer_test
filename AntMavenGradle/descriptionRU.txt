Сборка Java проектов: быстрый взгляд на Ant, Maven и Gradle

Умение работать с системами сборки нужно каждому Java-разработчику. На вебинаре вы узнаете о системах Apache Ant, Apache Maven и Gradle. Мы опробуем все три на простом Java-проекте: скомпилируем, запустим и протестируем его, собёрем jar файл. Нам понадобятся установленные и настроенные:

- Java SE DK 8 http://www.oracle.com/technetwork/java/javase/downloads/jdk8-downloads-2133151.html
- Apache Ant http://ant.apache.org/
- Apache Maven https://maven.apache.org/
- Gradle https://gradle.org/

Желательно, чтобы у вас уже было базовое знание Java и умение работать в командной строке. Материалы вебинара (исходные коды и сборочные сценарии) будут доступны по ссылке https://github.com/biblelamp/JavaExercises/AntMavenGradle
